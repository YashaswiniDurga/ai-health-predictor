# AI Health Risk Predictor

## 💡 Idea
An AI-based system that predicts health risks based on lifestyle and environmental factors like pollution.

## 🚀 Features
- Health score prediction
- Risk level detection
- Smart suggestions
- Pollution-based analysis
- AI chatbot assistant

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS
- JavaScript

## 🎯 Purpose
To promote preventive healthcare in smart cities.
